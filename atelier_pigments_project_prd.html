# Project PRD: Atelier Pigments - Premium Paint Website Design ## 🎯 Project Overview A complete, high-fidelity website design and interaction system for "Atelier Pigments," a premium paint company. The design focuses on an immersive, visually rich brand experience and lead generation. ## 🧱 Page Structure & Core User Flows 1. **Home Page:** Immersive hero with 3D elements, USPs, product previews, and project showcases. 2. **About Page:** Story-driven timeline, vision/mission, and leadership focus. 3. **Products Page:** Categorized catalog (Interior, Exterior, Industrial, Wood) with spec previews. 4. **Product Detail Page:** Deep dive into technical specs, usage guides, and documentation (TDS/MSDS). 5. **Dealer Locator:** Map-based search to convert interest into physical leads. 6. **Contact Page:** Structured lead generation form and global office locations. ## 🎨 Design System: "Atelier Pigment" * **Creative North Star:** "The Curated Atelier" — moving away from transactional e-commerce toward artisanal curation. * **Typography:** Newsreader (Serif) for headings to evoke heritage and craftsmanship; Manrope (Sans-serif) for functional UI and body text. * **Color Palette:** Inspired by natural pigments. Deep navy (#1B263B), terracotta accents (#944925), and cream-based neutrals (#FAF9F6). * **Spacing:** Consistent 8px/4px rhythm for modular layout structure. ## 🎞️ Interaction & Motion System * **Scroll-Driven Storytelling:** Smooth parallax and reveal animations as users traverse the narrative. * **3D Hero Interaction:** Simulated depth on the home page hero to create an immersive "Atelier" feel. * **Micro-interactions:** Hover states on cards, subtle scale effects on CTAs, and smooth transitions between page states. ## 📱 Responsiveness & Scalability * **Fluid Layouts:** Optimized for high-resolution desktop and tablet (sales tool use cases) while maintaining mobile-first principles. * **Future-Proofing:** Modular component structure allows for the addition of a "Color Visualizer" or "3D Material Viewer" in future iterations. ## 🧩 Reusable Component Library * `TopNavBar`: Sticky, high-blur navigation with primary "Order Sample" CTA. * `Footer`: Comprehensive site map with newsletter integration. * `ProductCard`: High-fidelity card with color swatches and specs. * `Timeline`: Interactive vertical or horizontal story component. * `LeadForm`: Clean, structured input system for inquiries.
